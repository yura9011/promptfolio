   PROMPTFOLIO - COMANDOS PARA DEPLOY
═══════════════════════════════════════════════════════════════

 PASO 1: Conectar con GitHub
───────────────────────────────────────────────────────────────
git remote add origin https://github.com/yura9011/promptfolio.git
git branch -M main
git push -u origin main


⚙ PASO 2: Activar GitHub Pages
───────────────────────────────────────────────────────────────
1. Ve a: https://github.com/yura9011/promptfolio/settings/pages
2. En "Source", selecciona: GitHub Actions
3. Guarda


 PASO 3: Verificar Deploy
───────────────────────────────────────────────────────────────
1. Ve a: https://github.com/yura9011/promptfolio/actions
2. Espera que termine el workflow (~2 minutos)
3. Tu sitio estará en: https://yura9011.github.io/promptfolio/


═══════════════════════════════════════════════════════════════
   SUBIR TUS PRIMERAS IMÁGENES
═══════════════════════════════════════════════════════════════

1⃣ Configurar Cloudinary:
   cp .env.example .env
   code .env
   (Agregar tus credenciales de https://console.cloudinary.com/)

2⃣ Preparar imágenes:
   mkdir my-images
   (Copiar tus imágenes .png/.jpg y crear archivos .txt)

3⃣ Subir:
   npm install
   npm run upload ./my-images

4⃣ Publicar:
   git add data/images.json
   git commit -m "Add my images"
   git push
